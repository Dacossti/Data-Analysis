## Overview

This repository contains a collection of **data analysis and data visualization projects** using different tools and technologies such as:

- Python
- Power BI
- R
- Streamlit
- R Shiny

The goal of this repository is to showcase **data analysis workflows, dashboard creation, and interactive applications** built as part of coursework and personal projects.
